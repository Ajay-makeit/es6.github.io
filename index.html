ES6 Rules

Template Literals

Template literals are string literals that allow for embedded expressions. They are denoted by the backtick (`) character, and can contain placeholders in the form of ${expression}.


const name = 'John';
const greeting = `Hello, ${name}!`;
console.log(greeting); // "Hello, John!"

Arrow Functions

Arrow functions are a shorthand syntax for writing functions in JavaScript. They have a shorter syntax compared to regular functions and do not bind their own `this` value.


  const add = (a, b) => a + b;
  console.log(add(1, 2)); // 3

Spread Operator

The spread operator allows an iterable (such as an array) to be expanded in places where multiple arguments (for function calls) or multiple elements (for array literals) are expected. It is denoted by three dots (...).


  const numbers = [1, 2, 3];
  console.log(Math.max(...numbers)); // 3

Destructuring Assignment

Destructuring assignment allows for extracting data from arrays and objects into distinct variables. It is often used in combination with the spread operator (...) to easily extract data from arrays and objects.


  const person = { name: 'John', age: 30 };
  const { name, age } = person;
  console.log(name); // "John"
  console.log(age); // 30
